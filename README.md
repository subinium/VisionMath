# VisionMath

Interactive Math & Physics Visualizations with Hand Tracking

![VisionMath Demo](demo.gif)

## Features

### Math Modes
- **Triangle Centers** - Visualize centroid, circumcenter, incenter, and orthocenter with interactive triangle types (Acute, Right, Obtuse)
- **Platonic Solids** - Explore 3D geometry with interactive rotation of the five Platonic solids

### Physics Modes
- **Vector Addition** - Interactive demonstration of vector addition with parallelogram law, magnitudes, angles, and cosine law
- **Pendulum** - Real-time pendulum simulation with accurate physics equations and adjustable parameters

## Controls

- **Hand Tracking**: Use your webcam to interact with visualizations using hand gestures
- **Mouse/Touch**: Click and drag to manipulate objects
- **Background Toggle**: Switch between camera view and dark mode

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- A modern web browser with webcam support

### Installation

```bash
git clone https://github.com/subinium/VisionMath.git
cd VisionMath
npm install
npm run dev
```

### Build for Production

```bash
npm run build
```

## Technologies

- **React** - UI framework
- **Vite** - Build tool and dev server
- **MediaPipe** - Hand tracking via webcam
- **Canvas API** - 2D/3D rendering
